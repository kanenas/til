If for some reason you can not auto update Google Chrome in your **Ubuntu 24.04.4 LTS** installation. Do the following:

#### 1. Download google-chrome-stable_current_amd64.deb

#### 2. Move to /tmp
`sudo mv ~/Downloads/google-chrome-stable_current_amd64.deb /tmp/`

#### 3. Install from /tmp
`cd /tmp`  
`sudo apt install ./google-chrome-stable_current_amd64.deb`

#### 4. Clean up (optional)
`sudo rm /tmp/google-chrome-stable_current_amd64.deb`
